📘 Online MCQ Examination System (Java):

📌 Description:

The Online MCQ Examination System is a console-based Java application that allows users to securely log in, update their profile, change passwords, and take multiple-choice tests. The system includes a timer feature with auto submission and ensures proper session handling with logout functionality.

🚀 Features:
🔐 User Login Authentication
👤 Update Profile (Name & Email)
🔑 Change Password
📝 Attempt MCQ-based Exam
⏱ Timer with Auto Submit
📊 Instant Score Display
🚪 Secure Logout

🛠️ Technologies Used:
Java (Core Java)
OOP Concepts
Console-based Interface
📂 Project Structure:
OnlineMCQExam.java

▶️ How to Run:
Copy the code into any Java IDE or Online Compiler
(VS Code / Eclipse / IntelliJ / OnlineGDB)
Compile the program:
javac OnlineMCQExam.java

Run the program:
java OnlineMCQExam
